# OMEM v1.8.0 架构重构实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将session_ingest从"碎片化提取"升级为"分类追加+语义归簇"，实现工作记忆降噪追加、情感记忆私密保留、用户偏好独立提取，并通过cluster tag体系和增量归簇替代K-Means全量覆盖。

**Architecture:** session_ingest三分类（WORK/EMOTIONAL/PREFERENCE），每类独立处理逻辑：WORK按项目×子主题追加（模仿EMOTIONAL模式），EMOTIONAL保持现有追加逻辑，PREFERENCE独立提取并存入画像。归簇改为增量语义匹配（session_id优先+语义相似度），加互斥锁防并发。新增cluster_llm用免费Qwen模型降低成本。

**Tech Stack:** Rust (axum 0.8, lancedb 0.27), TypeScript (OpenCode plugin SDK), LanceDB向量存储, DeepSeek V4 Flash (session_ingest LLM), Qwen3.5-4B via SiliconFlow (cluster/profile LLM, 免费)
